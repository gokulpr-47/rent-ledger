# RentLedger

RentLedger is a professional-grade rental management and credit ledger system built using the MERN stack.

It is designed for local rental shops to manage:

- Customer records
- Tool rentals
- Running credit accounts
- Discount management
- Bill grouping and credit closure
- Daily data backups

The system supports:

- Docker-based local deployment
- One-click update via GitHub
- Desktop application support using Electron
- Persistent MongoDB storage with safe volume management

Built with scalability and production-readiness in mind.
